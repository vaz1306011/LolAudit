# LolAudit

[中文](README.md) | [English](README_EN.md) | [日本語](README_JP.md)

**『リーグ・オブ・レジェンド (League of Legends)』** クライアント向けの補助ツールで、自動対戦承諾・自動再キュー・簡易 UI 操作を提供し、ランク戦や通常戦での繰り返し操作を減らします。

## [ダウンロード](https://github.com/vaz1306011/LolAudit/releases/latest)

## 機能 ✨

- **ワンクリックキュー**：設定したモードでロビーからワンクリックでキューを開始します。
- **自動対戦承諾**：設定した時間に従い、マッチ成立後に自動で承諾します。
- **自動再キュー**：キュー時間が想定を超えた場合に自動で再キューします。
- **禁止チャンピオン選択**：前回の選択に基づいて自動で禁止チャンピオンを選択します。
- **自動ロック**：タイマー終了前に自動でチャンピオンをロックします。

## 動作要件 📦

- **OS**:
  - Windows 10/11
  - macOS
- **Python**：3.10+
- **依存パッケージ**：
  - [PySide6](https://pypi.org/project/PySide6/)

## インストール方法 ⚙️

1. リポジトリをクローン：
   ```bash
   git clone https://github.com/vaz1306011/LolAudit.git
   cd LolAudit
   ```
2. 仮想環境を作成して依存関係をインストール：
   ```bash
   pipenv install
   ```
3. 実行：
   ```bash
   python lol_audit.pyw
   ```

## 使い方 🖥️

- **ワンクリックキュー**：「ワンクリックキュー」ボタンをクリックすると、選択したモードでキューを開始します。
- **キュー停止**：マッチング中はいつでも停止できます。
- **設定項目**：
  - `自動承諾`：設定した時間に従って自動で承諾します。
  - `自動再キュー`：キュー時間が想定を超えた場合に自動で再キューします。
  - `禁止チャンピオン選択`：前回の選択に基づいて自動で禁止チャンピオンを選択します。
  - `自動ロック`：タイマー終了前に自動でチャンピオンをロックします。
